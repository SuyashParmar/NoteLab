## ✨ Welcome to NoteLab

**NoteLab** is a lightning-fast, highly intuitive, Notion-style productivity workspace. Built with bleeding-edge technologies, NoteLab allows you to take notes, organize your life, and collaborate in real-time. Whether you are managing personal tasks or working with a team, NoteLab is your central hub for *everything*.

<br/>

## 🛠️ Tech Stack & Superpowers

This project isn't just another note-taking app; it's a powerhouse built with the best tools in the modern web ecosystem:

- ⚡ **Framework**: [Next.js 14](https://nextjs.org/) (App Router, Server Actions)
- 🗄️ **Database & Backend**: [Convex](https://www.convex.dev/) (Fully typed, Real-time sync)
- 🔐 **Authentication**: [Clerk](https://clerk.dev/) (Secure, modern auth)
- 📝 **Rich Text Editor**: [BlockNote](https://www.blocknotejs.org/) (Notion-style blocks!)
- 🎨 **Styling**: [Tailwind CSS](https://tailwindcss.com/) & [Radix UI](https://www.radix-ui.com/)
- ☁️ **File Uploads**: [EdgeStore](https://edgestore.dev/) (Fast & reliable media hosting)
- 🐻 **State Management**: [Zustand](https://zustand-demo.pmnd.rs/)

<br/>

## 🏃‍♂️ Get Started in 60 Seconds

Want to spin this up on your local machine? Let's go!

### 1. Clone the repository
```bash
git clone https://github.com/SuyashParmar/NoteLab.git
cd NoteLab
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Setup Environment Variables
Create a `.env.local` file and add the required keys (Check `.env.example` for reference).
You'll need keys for **Clerk**, **Convex**, and **EdgeStore**.

### 4. Run the Development Server
```bash
npm run dev
```

### 5. Fire up Convex
In a separate terminal, run:
```bash
npx convex dev
```

Boom! 💥 Open [http://localhost:3000](http://localhost:3000) and behold the magic.
